# Android Logcat

The Android Logcat package displays messages such as stack traces and [logs](https://docs.unity3d.com/ScriptReference/Debug.Log.html) from an Android device in the Unity Editor. You can use this information along with tools provided with the package to debug your application running on an Android device.
